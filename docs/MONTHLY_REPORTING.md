# Relatorio Mensal

Esta ferramenta deve apoiar, nao substituir, o criterio de marketing.

## Fluxo recomendado

1. Abrir o dashboard.
2. Ir a `Relatorio Mensal`.
3. Escolher o mes.
4. Rever:
   - novidades;
   - promocoes;
   - intensidade por marca;
   - Chicco vs mercado Wells online;
   - presenca online no fim do periodo.
5. Exportar Excel mensal se for preciso anexar detalhe.
6. Copiar o texto base.
7. Cruzar com sell-out e informacao comercial.
8. Editar o texto final antes de enviar.

## Dados Confidenciais

O dashboard esta publicado no GitHub Pages, por isso nao deve receber dados de sell-out nem informacao interna.

Fluxo seguro:

1. Manter o dashboard Wells apenas com dados publicos.
2. Usar sell-out apenas em analises locais e privadas.
3. Nao gravar sell-out em `data/`, `index.html` ou qualquer ficheiro versionado.
4. Levar para o email final apenas conclusoes que possam ser partilhadas.

## Linguagem recomendada

Usar formulacoes prudentes:

- "Foram identificadas online..."
- "Pode indicar..."
- "Sinal a acompanhar..."
- "A validar com sell-out..."
- "Nao equivale necessariamente a presenca em loja fisica..."

Evitar conclusoes fortes apenas com base no site:

- "A marca ganhou mercado..."
- "A Wells retirou de loja..."
- "A promocao impactou sell-out..."

## Notas Sobre Cardex

A vista de presenca online ajuda a perceber entradas, ausencias e possiveis alteracoes no sortido digital. Nao deve ser interpretada automaticamente como cardex de loja fisica.
